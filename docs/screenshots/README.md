# Screenshots

To add screenshots:
1. Run `npm run dev` in `frontend/`
2. Open http://localhost:3001 in Chrome
3. Log in with each role and take screenshots
4. Save as: `login.png`, `dashboard.png`, `tutor.png`, `students.png`, `reports.png`, `users.png`

Upload them to this folder, then the README badges will show them on GitHub.
